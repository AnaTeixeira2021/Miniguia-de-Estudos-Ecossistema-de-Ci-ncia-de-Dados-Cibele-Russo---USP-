# Miniguia-de-Estudos-Ecossistema-de-Ci-ncia-de-Dados-Cibele-Russo---USP-
Este caderno temático foi criado para centralizar e organizar os recursos educacionais da Profa. Dra. Cibele Russo (ICMC-USP). O objetivo é fornecer uma trilha de aprendizagem estruturada que abrange desde os fundamentos de Python até técnicas avançadas de Aprendizado de Máquina e Análise Multivariada.
